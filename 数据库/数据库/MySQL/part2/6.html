总结

  • 完整的select语句
select distinct *
from 表名
where ....
group by ... having ...
order by ...
limit star,count
  • 执行顺序为：
    • from 表名
    • where ....
    • group by ...
    • select distinct *
    • having ...
    • order by ...
    • limit star,count
  • 实际使用中，只是语句中某些部分的组合，而不是全部
